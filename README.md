## What is it?

这是一个使用.NET9基于WPF的OpenCVSharp实验工具实现。主要用于尝试在各种工业场景中的图像实验处理效果；

## Reference

| 功能             | 引用包                |
| ---------------- | --------------------- |
| OpenCV图像处理库 | OpenCvSharp4          |
| Mvvm框架         | CommunityToolkit.Mvvm |
| Material-UI控件  | MaterialDesignThemes  |
| HC-UI控件        | HandyControl          |
